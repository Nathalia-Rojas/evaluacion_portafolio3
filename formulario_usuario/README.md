# Formulario en Consola

Este programa solicita información básica al usuario y la almacena en variables
con distintos tipos de datos para demostrar el uso de `str`, `int`, `float` y `bool`.

## ¿Qué datos se ingresan?
- Nombre (cadena - `str`)
- Edad (entero - `int`)
- Estatura (flotante - `float`)
- Preferencia por la programación (booleano - `bool`)

## ¿Cómo ejecutar?
```bash
python formulario.py
